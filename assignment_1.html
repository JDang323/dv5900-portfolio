Assignment 1

The dataset used in this assignment is the World Happiness Report. The data is used in order to help policy makers make decisions and policies. The scores and ranking use data from the Gallup World Poll. The subset of data that is used in this assignment is the GDP per Capita and the Happiness Score that are represented on the X-axis and Y-axis respectively.

Basic Plot - Line Chart

This is a line chart showing how the GDP per capita and the Happiness Score are related. Using a line chart makes it easier to see how the increase in GDP per capita correlates to a higher happiness score within a country. This is done so by being able to see an upward trend. Line charts are excellent visual for showing trends. Changing the color of the line would not add anything because it is only one line. Keeping the blue default color is satisfactory given that the background is white and the line contrasts well with it making easy to see. The removal of the legend was done so to save on space clutter. The legend wasn't needed because there is no other color or line that needs to be distinguished and the line itself already correlates to the two axis.

Variation 1 - Scatter Plot

The scatter plot aboves shows the GDP per capita vs. the Happiness score. This plot shows the individual data points from the subset of data. Using the scatter plot makes it easy to see how all the data is distributed. For the same reason as the line chart, the legend was removed for decluttering and simplicity sake. The data points on the graph have also been set to 60% opacity - doing so allows us to see individual data points in more cluttered areas such as 1.2 - 1.4 GDP per capita. Using a scatter plot portrays the same message as the line chart but using a scatter plot shows that there are more data points from 1.0 GDP per capita and onwards. This suggests that out of the 156 countries in the data, there are more higher GDPs than lower ones.

Variation 2 - Scatter Plot with Trendline

The scatter plot above shows the GDP per capita vs. the Happiness Score with a trendline. Much like the previous scatter plot, it shows the individual data points which allows us to see how all the data is distributed. This difference in this chart is the trendline that was added. This added trendline allows for intrepretation for how the GDP per capita compares against the Happiness Score. If there is no data point for a specific GDP per capita or Happiness Score, the trendline can be used to predict which GDP correlates with a specific happiness score and vice versa. The trendline can also be used to analyze the existing data points. Anything above the trendline suggest that they have a higher happiness score for the given GDP per capita. If they are below the trendline, it suggests the happiness score is lower for the given GDP per capita. I have increased the opacity of the points back to 100% because the focus is moreso on the trendline and it's use.
