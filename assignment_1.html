Assignment 1

The dataset used in this assignment is the World Happiness Reprot. The data is used in order to help policy makers make decisions and policies. The scores and ranking use data from the Gallup World Poll.

This is a line chart showing how the GDP per capita and the Happiness Score are related. Using a line chart makes it easier to see how the increase in GDP per capita correlates to a higher happiness score within a country. This is done so by being able to see an upward trend. Line charts are excellent visual for showing trends. Changing the color of the line or anything would not add anything because it is only one line.
