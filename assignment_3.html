Assignment 3

This assignment is the Tableau Map. We were tasked to find a dataset that has a column with location and then load it in in Tableau and create a map-based visualization of the data. Then we have to create a 2nd visualization that is different from the geospatial one. Then we have to create a Dashboard that includes both new visualizations and link them so that both the map and new table respond to hovers or clicks.

This dataset is from Wine Reviews Dataset on Kaggle. The points/ratings represents the number of points WineEnthusiast rated the wine on a scale of 1-100 and the price represents the cost for a bottle of the wine. The map and bar chart will highlight which every country you are hovering over on both visualizations. The tooltip also includes this data. The color grading also gives you an idea of which countries have a higher rating or a higher cost per bottle.

One review was removed from the dataset because I could not find which country it was from.
